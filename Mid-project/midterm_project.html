3.3 Evaluation Metrics

Required Metrics:

1) Overall Accuracy

Accuracy = (Σ TP_i) / N

where:

  • TP_i = True Positives for class i
  • N = Total number of instances
  • i = 1, 2, ..., k (number of classes)

2) Per-class Precision, Recall, and F1-score

For each class i, we define:

  • True Positives (TP_i): Instances that truly belong to class i and are correctly predicted as class i.

  • False Positives (FP_i): Instances that do not belong to class i but are incorrectly predicted as class i.

  • False Negatives (FN_i): Instances that truly belong to class i but are incorrectly predicted as other classes.

  • Precision for class i:
Precision_i = TP_i / (TP_i + FP_i)
  • Recall for class i:
Recall_i = TP_i / (TP_i + FN_i)
  • F1-score for class i:
F1_i = 2 × (Precision_i × Recall_i) / (Precision_i + Recall_i)

3) Macro-averaged F1-score

Macro-F1 = (F1_1 + F1_2 + ... + F1_k) / k

where k is the number of classes.

4) Weighted-average F1-score

Weighted-F1 = Σ (w_i × F1_i)

where:

w_i = n_i / N
  • n_i = number of instances in class i
  • N = total number of instances
  • Σ w_i = 1
